# elm / 饿了么（伪） 🍔 😋

## 项目介绍

`elm-delivery` 项目是一套微信点餐系统

* 买家端包含了商品、评论、 商家介绍、订单、支付模块。

* 卖家端包含了类目增删、商品上下架、订单模块。

## 技术栈

* 前端 -- 尚未开发

JS + Node.js + Vue.js

* 后端

SpringBoot +  JPA + MyBatis + Redis + WebSocket + SpringCloud
